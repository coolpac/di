# Сайт-подарок для Дианы к 8 марта

## Контекст
Романтический подарок для Дианы Красковской к 8 марта. Белые розы, элегантный стиль, игра "Собери букет".

## Технологии
- Vite + React
- Framer Motion (анимации)
- HTML Canvas (лепестки + игра)
- Google Fonts: Playfair Display + Inter

## Секции
1. **Hero** — "С 8 марта, Диана" + падающие лепестки
2. **Поздравление** — текст + SVG-анимация розы
3. **Галерея пожеланий** — 4 карточки с 3D hover
4. **Игра "Собери букет"** — кликай на белые розы за 30 сек
5. **Финал** — анимация сердечек + подпись

## Палитра
- Белый (#FFFFFF), нежно-розовый (#FFF0F5), золотой (#D4AF37), тёмно-зелёный (#2D5016)
